Personal Portal – Overview

This portal is designed as a unified dashboard for personal projects, collaboration, and internal documentation. It integrates communication tools, meeting scheduling, project tracking, and cloud-based app deployment.

🧭 Navigation

The navigation bar links to core sections of the portal:

  • Dashboard: Overview of recent activity, quick links, and summary statistics.
  • Meetings: Integrated calendar and meeting manager with links to Zoom/Google Meet.
  • Chat: Real-time collaboration and internal messaging.
  • Files: Cloud-linked file management for documents and reference material.
  • Settings: User profile management, authentication, and API integrations.

⚙️ Deployment

The portal is containerized and deployed using AWS App Runner with Copilot CLI for automation. Deployment steps:

  1. Ensure Dockerfile builds successfully locally.
  2. Use copilot init to create the service.
  3. Run copilot deploy to push the container to AWS App Runner.
  4. Use copilot svc logs for live log review during rollout.

Copilot simplifies configuration by managing service creation, load balancing, and secret handling.

💻 Tech Stack

  • Frontend: HTML, CSS, Bootstrap, Vanilla JS
  • Backend: Python Flask (lightweight REST endpoints)
  • Deployment: AWS App Runner + Copilot CLI
  • Authentication: Flask-Login with session cookies
  • Data Storage: Cloud storage integration for shared files
  • Communication: CometChat Embed for live chat

🧩 Integrations

  • Google Calendar API – meeting scheduling and retrieval
  • CometChat – real-time messaging system
  • Pumble / Peb.io – optional legacy chat integrations
  • AWS Secrets Manager – secure storage of environment variables

🗂 Pages

🏠 Dashboard

Displays user-specific summaries, activity charts, and shortcuts to ongoing projects. Designed for quick status visibility on both desktop and mobile.

📅 Meetings

Integrates with Google Calendar API for automatic event synchronization. Upcoming events appear directly within the portal with join-link buttons.

📁 Files

Provides an interface to upload, download, and view project-related files. Files are stored securely in a connected cloud bucket and indexed by user ID.

💬 Chat

The Chat page now features a fully integrated real-time chat system powered by CometChat Embed. This enables all signed-in portal members to communicate instantly within a shared workspace.

  • Automatic Login & Identity: Users are automatically logged in to CometChat with their portal username and display name.
  • Portal Users Group: Everyone joins a shared public group called Portal Users—a central space for discussion, questions, and coordination.
  • Live Updates: Messages sync in real-time with no refresh needed.
  • Future Expansion: Support for private rooms and topic-based groups will be added later.

Tip: Keep chat discussions professional and focused on collaboration; history is persistent for reference.

👤 Profile & Settings

Manage account credentials, profile photo, and linked integrations. Options are provided for regenerating API keys and managing connected services.

🔒 Security Notes

  • Authentication handled via Flask-Login and hashed passwords.
  • Secrets stored securely with AWS Secrets Manager.
  • CometChat Auth Key temporarily hard-coded for testing—will migrate to secrets before production.
  • HTTPS enforced for all routes in deployment.
